Run the analysis locally

For many reasons it can be advantegeous to run the analysis locally on your own computer. Here we describe how to get the current databases (SWEET-Cat and exoplanet.eu) and to combine them in Python

Get the data

Before getting any further, please get the two databses below by clicking the buttons. Note that the data is updated on a weekly basis if any new planets are found, or homogeneous stellar parameters are added. If you already have a working python enviroment, chances are that you do not need to worry about any special requirements. However, we also make the list of Python packages available in the table below ready for download. Last you will need the script also provided in the table below.

SWEET-Cat Exoplanet.eu Requirements Script
SWEET-Cat.csv exoplanetEU.csv requirements.txt sweetcat.py

Combining the data

It is important to combine the two data sets if into one combined data set. This is done by comparing the stellar names. To do this use the script provided above.

Simply place the script, the requirement file, and the two data sets in the same working folder and run:
$ python sweetcat.py -h
to see some help text.

If you have any problems running the script, then you might have to install any missing dependencies:
$ pip install -r requirements.txt
If this does not resolve the issue, then feel free to contact us.
